# astral-journey for fzf

## Install

Source the variant in your shell rc:

```bash
# ~/.zshrc or ~/.bashrc
source ~/.local/share/astral-journey/tools/fzf/astral-blackhole.sh
```

Open a new shell and run `fzf` — it'll show a astral-amber border, astral spark prompt, and blue syntax highlighting in matches.

## Visual details

- Rounded border in astral-border color
- Label `astral` anchored top-left in astral-amber
- Prompt ` ✦  ` with spark emoji
- Pointer ` ❯ ` and marker ` ✓ ` for selections
- Match highlighting in astral-azure
- Info text in astral-fg-muted
